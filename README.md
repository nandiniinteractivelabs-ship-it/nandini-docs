## 🚀 What we do

**NANDINI Interactive Labs** builds software that helps brands increase engagement, generate leads, and improve customer retention through interactive gamified experiences.

We enable marketing teams to launch and manage campaigns without engineering effort — a campaign goes live on any website with a two-line snippet.

---

## 🎮 Our product: NANDINI

NANDINI is a **Gamification-as-a-Service (GaaS)** platform that lets companies create and deploy interactive marketing experiences in minutes — no code required.

### Interactive game library
- 🎡 Spin the Wheel
- 🎫 Scratch Cards
- 🎁 Mystery Boxes
- 🎄 Advent Calendars
- ❓ Quizzes & Trivia
- 📊 Polls & Surveys
- 👍 Swipe / This-or-That
- 🧠 Skill games — Memory Match, Tap-the-Target, Word Search
- 🏆 Leaderboards & competitions
- 🪙 Loyalty cards & reward systems
- 🔗 Refer-a-friend / viral loops
- 📝 Lead capture & registration

### More than single widgets
Campaigns aren't limited to one game. A built-in **flow engine** chains multiple
steps into branching journeys — e.g. *play → register to claim → reveal prize → refer a friend* —
with conditions, achievements, leaderboards, and a turns/credits economy across visits.

## 🏗️ Architecture (high level)

NANDINI is a multi-tenant SaaS platform built for embed-anywhere distribution:

- **Studio (frontend):** Next.js — no-code campaign builder with live preview
- **API (backend):** NestJS + MongoDB
- **Embeddable loader:** a **< 5 KB** vanilla-JS snippet, **Shadow-DOM isolated** so it never
  clashes with the host site's styles or scripts
- **Multi-tenant model:** organization → campaigns → players
- **Built-in localization:** English, Nepali, Hindi, Spanish (and extensible per campaign)
- **Event-driven analytics:** anonymous-first visitor tracking, retention cohorts, and lead attribution
- **Cloud-native deployment** behind a global CDN
