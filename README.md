## What is NANDINI?

**NANDINI** is a no-code **gamification platform** that helps brands turn passive website traffic into engagement, leads, and repeat customers. Marketing teams build interactive campaigns — spin-to-win wheels, quizzes, scratch cards, mystery boxes, loyalty programs and more — and embed them on any site (Shopify, WordPress, Webflow or custom) with a lightweight snippet.

Built by **NANDINI Interactive Labs**, NANDINI replaces static pop-ups and forms with interactive experiences that boost conversion rates, retention, and lead quality.

### More than single widgets — a flow engine

Campaigns aren't limited to one game. A built-in **flow engine** chains steps into branching journeys — for example *play → register to claim → reveal prize → refer a friend* — with conditions, achievements, leaderboards and a turns/credits economy that persists across visits.

## 🏗️ Architecture (high level)

NANDINI is a multi-tenant SaaS platform built for embed-anywhere distribution:

- **Studio (frontend):** Next.js — no-code campaign builder with live preview
- **API (backend):** NestJS + MongoDB
- **Embeddable loader:** a **< 5 KB** vanilla-JS snippet, **Shadow-DOM isolated** so it never clashes with the host site's CSS or scripts
- **Multi-tenant model:** organization → campaigns → players
- **Built-in localization:** English, Nepali, Hindi, Spanish (extensible per campaign)
- **Event-driven analytics:** anonymous-first visitor tracking, retention cohorts and lead attribution
- **Cloud-native deployment** behind a global CDN

## ❓ Frequently asked questions

### What is gamification-as-a-service (GaaS)?
Gamification-as-a-Service is software that lets companies add game mechanics — points, prizes, quizzes, spin-to-win wheels, leaderboards — to their marketing without building anything in-house. NANDINI delivers these as ready-to-launch, embeddable campaigns.

### How do you add a gamification campaign to a website?
You build a campaign in the NANDINI studio (no code), then paste a two-line snippet into your site. It works on Shopify, WordPress, Webflow or any custom site, and renders inside an isolated Shadow DOM so it never breaks your existing design.

### What types of marketing games does NANDINI support?
Spin the wheel, scratch cards, mystery boxes, advent calendars, quizzes, polls, surveys, swipe games, memory and skill games, leaderboards, loyalty cards, and refer-a-friend loops — and multi-step journeys that combine them.

### Is NANDINI no-code?
Yes. Marketers create, theme, translate and publish campaigns entirely through the studio — no developers required.

### Who is NANDINI for?
Marketing teams and agencies that want to increase engagement, capture leads and improve retention — across SaaS, e-commerce, education, fintech and D2C.

### What makes NANDINI different?
A lightweight (< 5 KB) embeddable loader, a full game library plus a flow engine for branching journeys, built-in multilingual support, and privacy-first, consent-based analytics — all no-code.
